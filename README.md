# MUROIGUNDAN2
## ムロイ軍団 「ムロイ(仮)」
予定管理アプリ

## エレベーターピッチ
"タスクの優先順位がわからない"という
大学生向けの"やりたいことがすぐ分かる"を実現する
"ムロイ（仮）"です。
これは、アプリが予定を管理し、既存のメモ帳・カレンダーアプリとは違って
優先順位が一目で分かります。
